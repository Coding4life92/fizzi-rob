
An interactive 3D soda product experience built with modern web technologies. This project combines Three.js rendering, Next.js performance, and Prismic CMS content management to create a visually engaging, customizable product page.

The site showcases a floating soda can rendered in 3D with smooth animations, scroll-based interactions, and dynamic content powered by a headless CMS.

## Tech Stack
* TypeScript – Type-safe development
* Next.js – React framework for server-side rendering and optimized performance
* Three.js – 3D rendering and WebGL graphics
* React Three Fiber & Drei – React abstractions for Three.js
* GSAP & ScrollTrigger – High-performance animations and scroll interactions
* Tailwind CSS – Utility-first styling
